# Digital Library Management System

## Project Overview
This is a web-based Digital Library Management System developed using PHP and MySQL. The system provides an easy-to-use interface for library administrators to manage books, students, book requests, and issue/return transactions efficiently.

## Features
- Add new books with details like name, author, publication, branch, price, quantity, and book photo.
- View reports of books, students, book requests, and issued books.
- Add and manage student records.
- Issue books to students and keep track of return status.
- Admin login/logout functionality for secure access.

## Technologies Used
- PHP
- MySQL
- HTML / CSS
- Bootstrap (optional for styling)
- Apache Server (XAMPP/WAMP)


## How to Run the Project
1. Clone or download this repository.
2. Import the database file (`library_db.sql`) into your MySQL server.
3. Configure database connection in `config.php`.
4. Run the project on a local server like XAMPP or WAMP.
5. Access the admin panel via `http://localhost/your_project_folder/admin/login.php`.
